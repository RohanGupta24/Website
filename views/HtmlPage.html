About

Hi! I'm Rohan, a freshman at Purdue University majoring in Computer Science. I'm a technology enthusiast ready to solve the world's problems through the most innovative software. I'm particularly interested in machine learning and data analytics, but I am willing to explore other realms of programming. Check out my Github for the types of programming projects I have done both inside and outside the classroom.

Personal Details

My very first programming inspiration came from a game design course at DigiPen Institute of Technology. Using C# and Unity, I was able to create a simple, yet entertaining game with just a few lines of code. My interest in programming continued the next summer when I took an Introduction to Object-Oriented Programming course at University of Washington.

My first internship opportunity was with UPS. I learned about agile development and the collaborative software developing environment. I participated in pair programming and helped develop an Android app that automated the process of loading and unloading packages. As a result of my contributions to this project, I was able to learn more about different software platforms, such as Visual Studio, SQL Server Management Studio, Xamarin, Couchbase, and Openshift. My internship gave me the opportunity to use MVC architecture and to build a Web Service that connected both the Native and web app. I also learned how to manage a database in SQL by creating stored procedures. I was able to successfully create a webpage using HTML, CSS, and Javascript that displayed random pieces of data from the database using graphs and charts from D3.js. Some of the languages used were C#, Java, Python, SQL, XAML, HTML, CSS, and Javascript

During my free time, I spend a lot of time exploring the areas of machine learning and data analytics as it relates to computer science. I created a lot of small programs using R and Python, and made a couple of Android and iOS apps this past summer.

I have participated in several hackathons and was able to complete a few projects. One of them is called Prescription Merge Conflict that detected the drug prescription that would conflict with a patient's health. Another projects allows teachers to share documents instantly on students' technological devices without hassle.

Some of the courses I have taken thus far include Introduction to Object-Oriented Programming, C Programming, Foundations of Computer Science, and Introduction to Game Design. I will be taking Computer Architecture, Data Structures and Algorithms, and Introduction to Web Development.

Some of the clubs that I am participating in include CS Undergraduate Student Board, Purdue Hackers, Hello World, SIGAI, SIGAPPS, and IEEE ROV.

ItineratR - HackIllinois

ItineratR is a website which plans the perfect vacation from you with just your budget and prospective dates. It calculates the possible destinations and upon choosing one generates the whole itenary for the place. I designed and developed the whole front-end with Angular and Material Design Components. I also used Google Maps API to plot data on the maps, and Google Maps Geocoding API to get coordinates from the data.

SinLess - BoilerMake IV

SinLess makes you a better person by punishing you when not doing well, rewarding otherwise. I worked solely on the Android app, implementing the SMS check for swearing, getting number of steps from built-in pedometer and designing the interface. SinLess won as one of the Top 10 hacks at BoilerMake IV, official hackathon of Purdue University.

Glass Half-Empty - WildHacks

Glass Half-Empty is a funky web app which uses custom Machine Learning model to detect if the glass of water is full and if the water is contaminated. I designed the whole front-end of the app in AngularJS, contributed to the server written in Node.JS and trained the Machine Learning model with Clarif.ai API.

Google Calendar for WebAssign

Google Calendar for WebAssign is a Chrome extension which adds URLs to add assignments, homeworks and tests on WebAssign to Google Calendar.

Counsl.me - WildHacks

Counsl.me is a location-based web service providing free online counseling sessions for mental health. It won the prize for "#HackHarrasment" for Best Hack against online harassment at ArchHacks, official hackathon of Washington University in St. Louis.

CelebFinder - HelloWorld

CelebFinder is Chrome browser extension which spots and recognizes any celebrity on screen with Microsoft's facial recognition API and displays his/her information from WikiPedia. It won the prize for "Best Web App" at HelloWorld hackathon.

2016 Election Visualizer

2016 Election Visualizer is realtime visualization of people tweeting about Donald Trump and/or Hillary Clinton around the world. The line height is proportional to the tweeting user's followers, and hence the tweet's reach. Try it live on electionvisualizer.com!

VGrep - MHacks

VGrep is an Android app which automatically generates notes from a video lecture. It uses Deepgram API to get audio transcripts and then uses Machine Learning algorithms to determine the important points in the lecture. The Android app allows users to jump to the important points in an intuitive UI.

Spexy - HackGT

Spexy is a remote monitoring and task management system for PCs and servers. It allows users to remotely check CPU Usage, disk usage, RAM usage and control process remotely from the web app. The PC client app written in Python works on both Windows and Linux.

Rexthor (beta) - HackMIT

Rexthor utilises facial emotion recognition to sense what and how the user is feeling the given day and sets the mood of lighting, music, temperature, TV shows, etc. accordingly. It also uses speech recognition to personalise for each user, whether he/she wants to skip to another song or change the settings of a device(air conditioner, TV, etc.)

Battlepedia for Pokémon Go

Battlepedia for Pokémon Go is a companion app for Pokémon Go with all the data, info and tools needed to win and defend gyms. The app was rejected from Google Play Store for containing copyrighted content(even though loads of other apps on Play Store use Pokémons without any problems), so I open sourced it on GitHub. For now the app is exclusively on Android, and the design is heavily inspired by Google's Material Guidelines.

GamesKart

GamesKart is an online shopping portal for digital games, gaming hardware and accessories. I worked with the Shopify API and themes to customize the e-commerce portal. I also Developed a light weight yet modern design inspired responsive corporate website, shown here.

Decrypt '15

Decrypt '15 was the online cryptic treasure hunt based case study by the Finance & Investment Cell of Hindu College, part of Delhi University. I was responsible for designing the whole website, front-end as well as the back-end which involved PHP, MySQL and AngularJS in addition to HTML5 and CSS3.

(c)ync 2015 App

Official app for (c)ync v2.0 the Inter School Technology Symposium of DPS Bhopal conducted by (c)ypher the Computer club of DPS Bhopal.

Le DPI Changer

Le DPI Changer allows you to update your screen DPI persistently in a single click and restore back easily. Le DPI Changer is the only app in the Play Store to be compatible with devices such as OnePlus 2, OnePlus X and devices running Android 6.0 Marshmallow. With over 100,000 downloads and average 4.1/5 rating on Google Play Store, the app is scaling pretty well.

(c)yntax

(c)yntax is the IT Bulletin of DPS Bhopal published by (c)ypher, the Computer Club of DPS, Bhopal. (c)yntax talks about how technology is important in this age and keeps its readers updated with latest news happening in the world of tech. I was the Creative Director and Head Graphic Designer of the first two volumes of the magazine, so basically everything from covers to index was designed by me on Adobe Photoshop and InDesign.

(c)ypher

(c)ypher is the official Computer Club of DPS Bhopal, which I co-founded. I contributed to website's design and development, which includes a members only admin panel, blog and some static webpages. Many of my other projects are results of this club.

Decypher

Decypher was an Online Cryptic Hunt, a part of the Inter School Technology Symposium (c)ync 2014 and (c)ync v2 2015. The website included user panel with a quiz like layout and an admin panel for the moderators to moderate the players. It also included some trick levels, some requiring editing the cookies or the web page source code. The development was done by me and a friend, while the designing was done by me alone with a few inputs from other group members.

(c)ync 2014

(c)ync was the first ever Inter School Technology Symposium held in the city of Bhopal organised by (c)ypher, the Computer Club of DPS Bhopal. The website includes a home page with details about the various events laid out linearly. Also there's a teacher's panel to register and track their students taking part in the symposium in an efficient UI. Both designing and development was done by me over the course of a few weeks. I also designed all the posters related to this event, which falls under Graphic Designing.

Cashback India

Cashback India is supposedly India's no. 1 cashback provider with over 500+ partnered brands to help you save extra when shopping online. I worked on the back-end of the website coding everything from user login and account to full fledged admin panel for controlling the website
