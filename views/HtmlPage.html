About

Hi! I'm Rohan, a freshman at Purdue University majoring in Computer Science. I'm a technology enthusiast ready to solve the world's problems through the most innovative software. I'm particularly interested in machine learning and data analytics, but I am willing to explore other realms of programming. Personal Details

My very first programming inspiration came from a game design course at DigiPen Institute of Technology. Using C# and Unity, I was able to create a simple, yet entertaining game with just a few lines of code. My interest in programming continued the next summer when I took an Introduction to Object-Oriented Programming course at University of Washington.

My first internship opportunity was with UPS. I learned about agile development and the collaborative software developing environment. I participated in pair programming and helped develop an Android app that automated the process of loading and unloading packages. As a result of my contributions to this project, I was able to learn more about different software platforms, such as Visual Studio, SQL Server Management Studio, Xamarin, Couchbase, and Openshift. My internship gave me the opportunity to use MVC architecture and to build a Web Service that connected both the Native and web app. I also learned how to manage a database in SQL by creating stored procedures. I was able to successfully create a webpage using HTML, CSS, and Javascript that displayed random pieces of data from the database using graphs and charts from D3.js. Some of the languages used were C#, Java, Python, SQL, XAML, HTML, CSS, and Javascript

During my free time, I spend a lot of time exploring the areas of machine learning and data analytics as it relates to computer science. I created a lot of small programs using R and Python, and made a couple of Android and iOS apps this past summer.

I have participated in several hackathons and was able to complete a few projects. One of them is called Prescription Merge Conflict that detected the drug prescription that would conflict with a patient's health. Another projects allows teachers to share documents instantly on students' technological devices without hassle.

Some of the courses I have taken thus far include Introduction to Object-Oriented Programming, C Programming, Foundations of Computer Science, and Introduction to Game Design. I will be taking Computer Architecture, Data Structures and Algorithms, and Introduction to Web Development.

Some of the clubs that I am participating in include CS Undergraduate Student Board, Purdue Hackers, Hello World, SIGAI, SIGAPPS, and IEEE ROV.
